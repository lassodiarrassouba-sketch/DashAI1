DIASCO

Assistant Android avec voix, caméra, code, images et création de sites.

Télécharger l’APK Android

Version 2.0.0 - Android 6.0 et plus

Installation

  1. Touchez le bouton de telechargement depuis votre telephone Android.
  2. Ouvrez le fichier APK telecharge.
  3. Autorisez l'installation depuis le navigateur si Android le demande.
  4. Lancez DIASCO. Le mode en ligne est déjà configuré.

Important

Cette version est distribuee hors Play Store. Android peut afficher un avertissement normal avant l'installation. Cette version utilise le backend HTTPS DIASCO déjà configuré. Les utilisateurs n’ont aucune URL à saisir.

Les cles OpenAI ne sont pas dans l'application. Elles restent uniquement cote serveur.
