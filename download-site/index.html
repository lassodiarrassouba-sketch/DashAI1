DashAI

Assistant IA Android avec voix, camera, code, formules et images.

Telecharger l'APK Android

Version 1.0.5 - Android 6.0 et plus

Installation

  1. Touchez le bouton de telechargement depuis votre telephone Android.
  2. Ouvrez le fichier APK telecharge.
  3. Autorisez l'installation depuis le navigateur si Android le demande.
  4. Lancez DashAI. Le mode en ligne est deja configure.

Important

Cette version est distribuee hors Play Store. Android peut afficher un avertissement normal avant l'installation. Cette version utilise le backend HTTPS DashAI deja configure. Les utilisateurs n'ont pas d'URL a saisir.

Les cles OpenAI ne sont pas dans l'application. Elles restent uniquement cote serveur.
