DashAI

Assistant IA Android avec voix, camera et images.

Telecharger l'APK Android

Version 1.0.3 - Android 6.0 et plus

Installation

  1. Touchez le bouton de telechargement depuis votre telephone Android.
  2. Ouvrez le fichier APK telecharge.
  3. Autorisez l'installation depuis le navigateur si Android le demande.
  4. Lancez DashAI, puis configurez l'URL du backend IA HTTPS.

Important

Cette version est distribuee hors Play Store. Android peut afficher un avertissement normal avant l'installation. Pour fonctionner en production, DashAI doit utiliser un backend public HTTPS, par exemple une API FastAPI deployee sur Render ou Cloud Run.

Les cles OpenAI ne sont pas dans l'application. Elles restent uniquement cote serveur.
