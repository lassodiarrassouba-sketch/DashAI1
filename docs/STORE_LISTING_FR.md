# Fiche Play Store - DashAI

## Nom

DashAI

## Description courte

Assistant IA vocal avec réveil privé, vision caméra et génération d'images.

## Description complète

DashAI est un assistant IA vocal conçu pour répondre aux questions, garder le contexte d'une conversation et aider l'utilisateur à comprendre ce qu'il voit.

Fonctions principales :

- conversation écrite ou vocale ;
- réveil vocal avec phrase "dis Diasco" ;
- vérification locale de la voix de l'utilisateur ;
- description d'images prises avec la caméra ;
- génération et affichage d'images sur demande ;
- backend IA sécurisé : les clés API restent hors de l'application.

DashAI utilise un backend HTTPS pour traiter les demandes IA. L'empreinte vocale utilisée pour le réveil privé reste stockée localement sur l'appareil.
