# Consignes Codex — PLP Analyse Régionale

- Lire `README.md` avant toute modification.
- Travailler sur une copie du `.xlsm` et produire une nouvelle version numérotée.
- Ne jamais annoncer PDF/impression comme réparés sans test réel dans Excel Windows.
- Ne pas utiliser LibreOffice pour réenregistrer le classeur.
- Conserver toutes les fonctions existantes : import CSV/Excel, collage, recherche, couleurs, statistiques, graphique, export Excel.
- Spécifications immuables : capacité 3 000 ; admission si moyenne >= 12 ; cinq régions ; pourcentages visibles sur le camembert.
- Ne jamais commiter de données personnelles. Utiliser uniquement `samples/liste_anonymisee.csv`.
- Restaurer systématiquement les états Excel après toute macro, y compris après erreur ou annulation.
- Le correctif prioritaire doit utiliser une feuille rapport dédiée et fournir des diagnostics VBA complets.
